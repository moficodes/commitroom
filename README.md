# Commitroom
## Try it out!
Live demo is available here: [https://commitroom-598464211339.us-west2.run.app](https://commitroom-598464211339.us-west2.run.app)


A collaborative, real-time accountability room designed to enforce focus through public commitment. 

## Productivity Value
Commitroom leverages the psychological power of public accountability. By broadcasting your current task and its duration to the "Public Square," you create social pressure to stay on track. Built-in Pomodoro timers (focus/break protocols) enforce structured work sessions, preventing burnout while maximizing deep work output.

## How to Run & Test
This project uses Vite and requires Node.js.

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the provided local URL in your browser. Enter a username to join the room and broadcast your first commitment.
